# 杭州潜水鱼电子商务有限公司官网（单页版）

这是一个简单、可直接上线的公司官网单页项目，包含：

- 公司名称与品牌头图
- 公司简介
- 主营业务模块
- 联系方式
- 在线留言表单（前端校验 + 提交提示）

## 目录结构

- `index.html`：页面结构
- `styles.css`：页面样式
- `script.js`：页面交互

## 本地运行方式

### 方式一：直接双击打开

直接在文件管理器中打开 `index.html` 即可预览。

### 方式二：使用本地 HTTP 服务（推荐）

在项目目录执行：

```bash
python3 -m http.server 8080
```

浏览器访问：

<http://localhost:8080>

## 可选修改项

- 把 `index.html` 里的联系电话、邮箱、地址改为真实信息
- 替换文案与业务内容
- 如需真正接收留言，可接入后端接口（如 Node.js / PHP / 云函数）

## 部署建议（最简单）

- 直接上传到静态托管平台（如 Vercel、Netlify、GitHub Pages）
- 或上传到公司服务器（Nginx）作为静态站点

## 域名要不要买？

- 不买也能用：平台会给一个二级域名（例如 `xxx.vercel.app`）
- 要正式对外展示，建议买：更专业、方便宣传、方便客户记忆
- 公司官网通常建议购买 `.com` 或 `.cn`

## 域名购买与绑定（简版）

1. 在域名平台注册（阿里云、腾讯云、华为云等）
2. 搜索并购买你要的域名（例如 `qianshuiyu.com`）
3. 到部署平台（如 Vercel）添加自定义域名
4. 在域名平台添加 DNS 解析记录（按平台提示填写）
5. 等待生效（通常几分钟到 24 小时）

完成后，访问你的域名就能打开网站。
